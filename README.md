# 🩺 ElderEase AI

An AI-powered healthcare management platform designed to help elderly users manage medicines, appointments, AI-assisted health queries, and health reports in one secure application.

## 🌐 Live Demo

- **Frontend:** https://elder-ease-ai-neon.vercel.app
- **Backend:** https://eldereaseai.onrender.com

---

## ✨ Features

### 🔐 Authentication
- User Registration & Login
- JWT Authentication
- Protected Routes

### 💊 Medicine Management
- Add, Edit & Delete Medicines
- Mark Medicines as Taken
- Medicine Reminder Status
- Low Stock Alerts
- Refill Medicines

### 📅 Appointment Management
- Add, Edit & Delete Appointments
- Mark Appointments as Completed
- Upcoming Appointment Tracking

### 🤖 AI Health Assistant
- AI-powered health guidance
- Elder-friendly responses
- Simple wellness advice

### 📊 Dashboard
- Personalized Greeting
- Medicine Summary
- Appointment Summary
- Progress Tracking
- Recent Activity

### 📄 Reports
- PDF Report Generation
- Medicine Summary
- Appointment Summary
- Profile Information

### 👤 Profile
- Update Personal Information
- Persistent User Data

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/vrinda-j-devadiga/ElderEaseAI.git
```

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file inside the backend folder with:

```env
PORT=
MONGO_URI=
JWT_SECRET=
OPENROUTER_API_KEY=
```
